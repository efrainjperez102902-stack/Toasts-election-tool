# Toast's Little Election Tool

Static election simulator site with a US homepage and individual state pages.

## Entry Point

- `index.html`

## Deploy

This folder is ready for static hosting such as GitHub Pages.

1. Push the folder contents to a GitHub repository.
2. Enable GitHub Pages from the repository root.
3. Keep `.nojekyll` in place so the files are served as-is.

## Notes

- Most pages are self-contained HTML files.
- Some pages load local data bundles in nearby `.js` files.
- A few pages also load remote libraries such as D3 or TopoJSON from CDN.
